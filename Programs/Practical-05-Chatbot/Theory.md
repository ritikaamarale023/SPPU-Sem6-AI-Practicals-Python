# Practical 05 - Elementary Chatbot

## Aim
To develop an elementary chatbot for customer interaction application using Python.

---

## Theory

A chatbot is an Artificial Intelligence application that interacts with users through text or voice.

An elementary chatbot works using predefined rules and responses.

The chatbot takes user input and provides suitable replies based on matching conditions.

This chatbot is developed using:
- Conditional statements
- String matching
- Looping

---

## Applications
- Customer support
- Online help desk
- Banking assistance
- E-commerce websites
- Educational systems

---

## Algorithm
1. Start chatbot
2. Take user input
3. Compare input with predefined conditions
4. Display suitable response
5. Repeat until user exits
