# Practical 03 - Greedy Algorithm using Selection Sort

## Aim
To implement Greedy Search Algorithm using Selection Sort in Python.

---

## Theory

Greedy Algorithm is an approach that makes the best possible choice at each step to find the overall optimal solution.

Selection Sort follows the greedy technique by repeatedly selecting the minimum element from the unsorted part and placing it at the correct position.

At every iteration:
- Find smallest element
- Swap with current position
- Repeat for remaining array

---

## Applications
- Data sorting
- Scheduling problems
- Optimization problems
- Resource allocation

---

## Algorithm
1. Start from first element
2. Find minimum element in remaining array
3. Swap minimum element with current element
4. Move to next position
5. Repeat until array is sorted
