# Practical 06 - Expert System Help Desk Management

## Aim
To implement an Expert System for Help Desk Management using Python.

---

## Theory

An Expert System is an Artificial Intelligence application that uses knowledge and rules to solve problems like a human expert.

The system contains:
- Knowledge Base
- Rules
- Inference Engine

This help desk management system provides solutions for common technical issues using predefined conditions.

---

## Applications
- Customer support
- Medical diagnosis
- Technical troubleshooting
- Banking systems
- Online support systems

---

## Algorithm
1. Start expert system
2. Take user problem as input
3. Match input with predefined rules
4. Display suitable solution
5. Repeat until user exits
